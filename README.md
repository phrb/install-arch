### Install-arch

This is my arch configuration script.

#### Dependencies

Install arch linux.

Install git:

```
sudo pacman -S git
```

[Configure sudo](https://wiki.archlinux.org/index.php/sudo#Configuration)
for your user.

#### Installing

Run:

```
git clone git@github.com:phrb/install-arch.git
cd install-arch && ./install.sh
```
